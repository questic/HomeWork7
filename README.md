# HomeWork7

*How to build:*
1) Go to the directory with source code
2) Make directory for build:
```
    mkdir build
```
3) Go to the build directory:
```
    cd build
```
4) Run cmake:
```
    cmake ..
```    
5) Run make:
```
    make
```    
6) Run binary file:
```
    ./HomeWork7
```  

Or you can use your IDE. <br />
